# Costco_Categorizer

Use the following command to clone the repo from the location you want the code to be stored:
```https://github.com/sameerdshar/Costco_Categorizer.git```

Move the image of the costco receipt to:
```{your_source_directory}/costco_categorizer/src_img/```

Run the code using the following commands:
```
cd {your_source_directory}/costco_categorizer
python3 -m src.receipt.main
```
